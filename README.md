# youtube-download
Download youtube video in the best quality(4k)

There are some different ways to accomplish the task

## 下载类型

### 720p及以下
  直接下载最高质量视频，最高质量表示的是音频和视频综合质量最高
### 1080p及以上 
  单独下载最高质量音频和最高质量视频，采用不同方式合并音频和视频 
#### ffmpeg合并
#### youtubeDL自动调用ffmpeg合并
